# EcoShop Seasonal Performance & Churn Analysis

## Project Overview

This project analyzes customer purchasing behavior using RFM analysis, SQL, Power BI, and Machine Learning to predict customer purchase probability.

## Tools Used

* Python
* Pandas
* NumPy
* SQL (MySQL)
* Power BI
* Scikit-learn
* GitHub

## Workflow

1. Data Cleaning
2. SQL Analysis
3. RFM Segmentation
4. Power BI Dashboard
5. Machine Learning Model
6. Model Evaluation
7. Model Saving

## Business Insights

* Identified high-value customers using RFM analysis.
* Built an interactive Power BI dashboard.
* Predicted customer purchase probability using Logistic Regression.
* Evaluated the model using Accuracy and F1-Score.
* Saved the trained model for future use.

## Project Deliverables

* Python Notebook (.ipynb)
* SQL Scripts (.sql)
* Power BI Dashboard (.pbix)
* Trained Model (.pkl)
* README Documentation
